# Coding Class 2026 Signup Page

Static GitHub Pages site for collecting interest in a coding class covering Visual Studio, GitHub Copilot, Codex, iOS, Android, web apps, and ESP32 live demonstrations.

## Files

- `index.html` - the public signup page.
- `styles.css` - responsive layout and visual design.
- `script.js` - client-side form submission and validation.
- `assets/coding-class-hero.png` - generated local hero image used by the page.
- `docs/GITHUB_PAGES.md` - deployment steps.
- `docs/FORM_SETUP.md` - form endpoint setup steps.

## Local Preview

Open `index.html` directly in a browser. No build step is required.

## Before Publishing

1. Confirm the FormSubmit activation email for `jonasamidu@gmail.com` after the first test submission.
2. Update any course wording, contact details, or privacy language.
3. Push the repository to GitHub.
4. Enable GitHub Pages for the repository.

GitHub Pages can host the page, but it cannot store form submissions on its own. This page is wired to FormSubmit for email delivery.
